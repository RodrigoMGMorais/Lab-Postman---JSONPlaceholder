# 📡 Teste de API REST com Postman
Projeto prático desenvolvido para aprender e testar requisições HTTP em uma API pública, usando a ferramenta Postman.
---

## 🛠️ Ferramentas utilizadas
- **Postman**: para criar, enviar e testar requisições
- **JSONPlaceholder**: API pública gratuita para testes e simulações

---

## 📋 Funcionalidades implementadas
| Método | Ação |
|---|---|
| `GET` | Listar todos os usuários / buscar um usuário por ID / buscar publicações |
| `POST` | Criar um novo registro |
| `PUT` | Atualizar **todos os campos** de um registro existente |
| `PATCH` | Atualizar **apenas campos específicos** |
| `DELETE` | Excluir um registro |

---

## ✅ Diferenciais
- Uso de **variáveis** para manter a URL organizada e reutilizável
- **Testes automáticos** para validar o status da resposta e os dados retornados
- Estrutura organizada e documentada
